# Cloud-based Chatbot
An implementation of RAG-capable smart chat assistant with n8n cloud and LLMs (Google Gemini, Ollama and Mistral)

## Capabilities
This project is a chatbot for customer support. It may be used as an internal team-communication tool as well.

### Tech tooling
It is built upon:
- n8n
- LLM
- Google Gemini
- Ollama
- Mistral
- RAG implementation
